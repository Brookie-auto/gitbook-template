# 🎵 Concerts & Events

## Browse Available Events

The **Concerts & Events** page is your hub for discovering upcoming shows across the Philippines.

### Event Information

Every event listing includes:

| Field          | Description                                  |
| -------------- | -------------------------------------------- |
| Event Name     | Name of the concert or festival              |
| Venue          | Location of the event (e.g., MOA Arena)      |
| Event Date     | When the event takes place                   |
| Selling Date   | When tickets go on sale                      |
| Platform       | SM Tickets or Ticketnet                      |
| Status         | Active or Inactive                           |

### How Events Work

1. Our team adds events as they are announced
2. Events become visible once marked as **Active**
3. Selling dates tell you when tickets become available
4. You can submit a booking request before or on the selling date

{% hint style="info" %}
Bookmark events you're interested in and check back as the selling date approaches for the best chance of securing tickets.
{% endhint %}
