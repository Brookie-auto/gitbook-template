# 🗺️ Seat Selection

## Interactive Venue Maps

For supported venues, TicketsPH provides interactive seat maps so you can choose your preferred section.

### How to Use the Seat Map

1. Click **"Secure Ticket"** on an event
2. The seat map modal will appear showing the venue layout
3. **Hover** over sections to see pricing details
4. **Click** on your preferred section to select it
5. Confirm your selection and proceed with booking

### Supported Venues

| Venue       | Seat Map Available |
| ----------- | ------------------ |
| MOA Arena   | Yes                |
| More coming | Soon               |

### Section Information

Each section on the map shows:

* **Section name** (e.g., VIP, Lower Box, Upper Box, General Admission)
* **Price range** for that section
* **Availability** status

{% hint style="info" %}
Not all events have seat maps available. For events without a seat map, you can specify your preferred section in the **notes** field when booking.
{% endhint %}
