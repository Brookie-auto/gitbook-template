# Proxies

## Setting Up Your Proxies

Proxies are essential for running tasks efficiently.

### Supported Formats

```
ip:port
ip:port:username:password
username:password@ip:port
```

### Adding Proxies

1. Navigate to the **Proxies** tab
2. Click **Add Proxy List**
3. Give your list a name
4. Paste your proxies (one per line)
5. Click **Save**

### Proxy Types

| Type        | Speed   | Recommended For    |
| ----------- | ------- | ------------------ |
| Residential | Medium  | General use        |
| ISP         | Fast    | High-demand events |
| Datacenter  | Fastest | Testing            |

{% hint style="warning" %}
Avoid using free or public proxies — they are unreliable and may get your tasks blocked.
{% endhint %}
