# Settings

## Configuring Your Settings

### General Settings

* **Theme** — Switch between light and dark mode
* **Notifications** — Enable or disable desktop notifications
* **Auto-Update** — Keep the software up to date automatically

### Task Settings

* **Default Retry Delay** — Time between retry attempts (in ms)
* **Max Retries** — Maximum number of retry attempts per task
* **Timeout** — How long to wait before a task times out

{% hint style="info" %}
Lower retry delays are faster but may trigger rate limits.
{% endhint %}
