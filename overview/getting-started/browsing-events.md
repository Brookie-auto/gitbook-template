# Browsing Events

## Finding Concerts & Events

### Navigate to Concerts & Events

Click **"Concerts & Events"** in the navigation bar to see all active events.

### What You'll See

Each event card displays:

* **Event thumbnail** — promotional image
* **Event name** — the concert or festival name
* **Venue** — where the event takes place
* **Event date(s)** — when the event happens
* **Selling date(s)** — when tickets become available for purchase
* **Platform** — SM Tickets or Ticketnet

### Finding the Right Event

Browse through active events and click on any event card to view more details including:

* Full event description
* Interactive seat map (when available)
* Section pricing
* Available booking slots

{% hint style="info" %}
Events are only shown while they are **active**. Check back regularly for newly added events!
{% endhint %}
