# Device & Server Setting

## Configuring Your Device

### Minimum Requirements

| Component | Requirement     |
| --------- | --------------- |
| OS        | Windows 10+     |
| RAM       | 4 GB minimum    |
| CPU       | Dual-core       |
| Network   | Stable internet |

### Server Settings

Configure your server connection in **Settings → Server**.

* **Server Region** — select the closest region for best performance
* **Connection Mode** — choose between direct or proxy-routed connections
